=== Subject Identifier Constraint ===

The 'Subject Identifier Constraint' is used to  demand for a subject identifier of topics using the containing
'Topic Type'. It is possible to constraint the number of identifier and also the form, using a regular expression.

.Property Page 'Subject Identifier Constraint'
image::images/si_constr_pp.png[]

The following attributes can be modified:

* *cardMin:* The minimum number of identifier with the given type.
* *cardMax:* The maximum number of identifier with the given type.
* *reg. exp:* The regular expression specifying the syntax of the subject identifier.
