=== Documentation ===

Every element of the schema has a additional properties, which are used for documenting.
The properties are:

* *comment:* Used to comment a schema element. This may be used by persons reviewing the schema.
* *description:* Used to describe the element. This is used by the author of the schema.
* *see also:* Used to add a location, where to get more information. This may be any kind of resource 
but mostly will be a URL.

.Property Page of documentation
image::images/documentation_pp.png[scaledwidth="100%"]

To modify these properties use the tab lablled 'Documentation', which is part of the *Property Details View* of 
almost every element.

NOTE: When you export the schema the documentation is exported according to the TMCL draft.
